# Bridging Tokens

## Overview

This guide explains how to bridge tokens between different chains in the Curve ecosystem.

## Prerequisites

- A Web3 wallet (MetaMask, WalletConnect, etc.)
- Tokens to bridge
- Gas fees for the source and destination chains

## Steps to Bridge Tokens

1. Connect your wallet
2. Select the source and destination chains
3. Choose the token you want to bridge
4. Enter the amount
5. Confirm the transaction

## Supported Tokens

- ETH
- USDC
- USDT
- DAI

## Fees

Bridging fees vary depending on:
- Network congestion
- Token type
- Amount being bridged

## Troubleshooting

If you encounter any issues while bridging tokens, please check:
- Your wallet connection
- Network settings
- Gas fees
- Token approvals
