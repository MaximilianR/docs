# Introduction

Curve is a decentralized, permissionless protocol for creating and accessing on-chain liquidity. Anyone can launch a pool, deploy a lending market, or integrate with its infrastructure — no approvals required.

This guide is for stablecoin issuers, protocols, DAOs, and builders looking to grow on-chain liquidity and build composable financial products.
